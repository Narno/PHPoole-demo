PHPoole demo

This PHPoole demo is a simple blog, based on the Hyde theme.

Installation

$ composer create-project narno/phpoole-demo

Usage

With PHPoole CLI:

$ cd phpoole-demo
$ phpoole build -s

Or with the included PHP script:

$ cd phpoole-demo
$ php build.php
